# HelpX

HelpX is an AI-assisted system designed to improve communication accuracy in emergency helplines like 1092.

## Problem
Citizens often speak in different languages and dialects during emergencies, which can lead to misunderstandings and incorrect responses.

## Solution
HelpX ensures correct understanding before action by introducing a verification step in communication.

## Features
- AI-based understanding of user input
- Sentiment and risk detection
- Verification loop (Correct / Partially Correct / Incorrect)
- Emergency SOS button for instant escalation
- Human agent takeover for critical situations

## How It Works
1. User provides input (text/voice simulated)
2. AI analyzes and generates summary, sentiment, and risk
3. System asks for verification
4. If confirmed → proceed
5. If incorrect or high risk → escalate to human agent
